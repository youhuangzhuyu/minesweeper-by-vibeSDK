# 🎮 Minesweeper Fresh (扫雷现代版)

一个基于 React 和 Tailwind CSS 开发的现代、明亮风格的网页版扫雷游戏。

## ✨ 特性

- **现代 UI**: 采用 Plus Jakarta Sans 字体和清爽的蓝白配色方案。
- **全响应式设计**: 游戏面板会根据屏幕大小自动缩放，无需滚动即可畅玩各种难度。
- **三种经典难度**: 
  - **Easy (初级)**: 9x9, 10个雷
  - **Medium (中级)**: 16x16, 40个雷
  - **Hard (高级)**: 30x16, 99个雷
- **专业游戏逻辑**:
  - **首击保护**: 第一次点击永远不会踩雷。
  - **双击联动 (Chording)**: 当周围旗帜数满足数字要求时，双击数字可快速扫除。
  - **自动展开**: 递归展开空白区域。

## 🕹️ 操作指南

- **左键单击**: 揭开方格。
- **右键单击**: 插上或拔掉旗帜。
- **双击数字**: 在已插好足够旗帜的情况下，快速揭开周围所有非旗帜方格。
- **Settings (齿轮按钮)**: 展开难度选择面板。
- **New Game (刷新按钮)**: 随时重开一局。

## 🛠️ 技术栈

- **前端框架**: React 18
- **样式**: Tailwind CSS
- **图标**: Lucide React
- **动画**: Framer Motion
- **构建工具**: Cloudflare Worker Bundler

## 🚀 部署与运行

该项目已配置为在 Cloudflare Workers 上运行。

1. 修改 `public/` 下的文件进行开发。
2. 使用 `deploy_space` 命令提交并部署到空间。
3. 访问部署后的预览 URL 即可开始游戏。
